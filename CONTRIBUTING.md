# Contributing to SmartPage Toolbox

Thank you for your interest in contributing to SmartPage Toolbox!

## Development Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/wveck/smartpage-toolbox.git
   cd smartpage-toolbox
   ```

2. Load the extension in Chrome:
   - Open `chrome://extensions`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the repository folder

3. Make your changes and test locally

## Code Style

- Use vanilla JavaScript (ES modules)
- Follow existing code structure and naming conventions
- Add comments for complex logic
- Keep functions focused and modular

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description

## Reporting Issues

Please use GitHub Issues to report bugs or suggest features. Include:
- Chrome version
- Extension version
- Steps to reproduce
- Expected vs actual behavior

Thank you for contributing!
